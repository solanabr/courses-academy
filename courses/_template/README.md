# `_template` — the copy-me course

This folder is a **complete, valid example** of every piece a course can contain: a `course.yaml`, a `slots.lock.json`, and lessons demonstrating each block type (prose, video, quiz, code in TypeScript and Rust, wallet-funding, program-explorer, deployed-program-card, and a reflection).

## Use it

```bash
cp -r courses/_template courses/<your-slug>
```

Then edit `course.yaml` (change `id`, `slug`, `title`, `creator`, `modules`), rename the lesson folders, and replace the block content. Rename the lesson `id`s and the `course.id` — every id must be unique and correctly prefixed.

Don't hand-edit `slots.lock.json`; it's regenerated by the tooling.

## It is never published

The template is **not** a real course and never appears on the platform — it's only here as an example to copy. It *is* checked by CI, on purpose: that keeps it a guaranteed-valid reference that can't quietly go stale.

See [`../README.md`](../README.md) for the full course format and rules.
