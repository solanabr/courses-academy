# What is Solana?

Solana is a high-performance blockchain supporting builders around the world creating crypto apps that scale.

## Key Features

- **Speed**: Solana can process 65,000+ transactions per second
- **Low Cost**: Average transaction cost is $0.00025
- **Energy Efficient**: Solana uses a Proof-of-Stake consensus mechanism
- **Developer Friendly**: Programs can be written in Rust, C, or C++

## How Solana Works

Solana uses a unique combination of **Proof of History (PoH)** and **Proof of Stake (PoS)** to achieve high throughput.

### Proof of History

PoH creates a historical record that proves events occurred at a specific moment in time. Think of it as a cryptographic clock that helps the network agree on the order of transactions without waiting for other validators.

### The Solana Runtime

Solana programs (smart contracts) run inside the **Sealevel** runtime, which allows thousands of smart contracts to run in parallel. This is one of the key innovations that enables Solana's speed.
